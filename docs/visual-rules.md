# Visual Rules

## Layers

- Idea Layer: `y = 2`
- Knowledge Layer: `y = 0`
- Source Layer: `y = -2`
- Project Layer: `x = -4`

Project nodes live on a side layer and are not aligned to the three horizontal planes.

## Colors

- Source: blue
- Knowledge: green
- Idea: warm rose/orange
- Project: purple

`Project -> Source` targets are marked as newly generated sources with a dotted halo.

## Interaction

The MVP supports Global View, Project Focus, and a right-side detail panel. Project Focus shows:

```text
Knowledge -> Project <- Idea
              |
              v
            Source
```
